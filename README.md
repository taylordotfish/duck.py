duck.py
=======

A [Weechat] plugin port of duck.vbs ([distributed here][duck.vbs]) from
[NO5 IRC].

[Weechat]: https://weechat.org/
[duck.vbs]: https://fioresoft.net/download3.php
[NO5 IRC]: https://github.com/fioresoft/no5irc
